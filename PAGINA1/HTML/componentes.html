  4. Desplegar en producción

Notas

Skeleton aplica márgenes y tipografía limpia a listas, párrafos y encabezados. No necesitamos definir estilos adicionales.

Ejemplo de código


// Petición básica desde el cliente (JavaScript)
fetch('/api/usuarios')
  .then(respuesta => respuesta.json())
  .then(datos => console.log(datos));

Fíjate cómo Skeleton ya le da estilo al elemento <code> y al bloque <pre> sin que tengamos que escribir más CSS.

Imagen responsiva

La clase .u-max-full-width hace que la imagen no se desborde del contenedor y se adapte al ancho disponible.
