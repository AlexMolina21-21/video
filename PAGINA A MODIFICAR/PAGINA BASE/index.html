¿Qué es un sistema web?

Un sistema web es una aplicación que funciona sobre una red (como Internet) y se accede mediante un navegador. Está compuesto por un lado cliente (front-end), un servidor (back-end) y, normalmente, una base de datos.

Skeleton nos ayuda a estructurar el contenido de forma responsiva usando filas (.row) y columnas (.columns).

Características

  • Acceso desde navegador
  • Arquitectura cliente-servidor
  • Comunicación mediante HTTP/HTTPS
  • Uso de HTML, CSS y JavaScript

Comparación rápida

Elemento Función Tecnologías típicas
Front-End Interfaz de usuario HTML, CSS, JS
Back-End Lógica del sistema Node, PHP, Java, Python
Base de datos Almacenamiento de datos MySQL, PostgreSQL, MongoDB

La clase .u-full-width es parte de Skeleton y hace que la tabla use todo el ancho disponible dentro del contenedor.
